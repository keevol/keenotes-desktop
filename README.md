# 常见问题（You Might Want to know)

## dmg文件损坏（dmg file damaged/warning/error）

如果你遇到如下问题：

![](images/dmg-damaged.png)

可以这样: `sudo xattr -d com.apple.quarantine /Applications/KeeNotes\ Desktop.app`

应该是Mac的安全机制导致，可能是因为我本地编译发布，没有做认证的缘故。
